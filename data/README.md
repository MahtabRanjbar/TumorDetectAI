# Brain Tumor Detection Dataset

To run the brain tumor detection project, you need to download the dataset from the Kaggle link provided below and organize it in the following directory structure:

```
      data
      ├── yes
      │   ├── image1.jpg
      │   ├── image2.jpg
      │   └── ...
      └── no
          ├── image1.jpg
          ├── image2.jpg
          └── ...
```
